# 🧩blockAi🧩

## 👀 프로젝트 소개
공인인증서를 사용한 본인인증이 번거롭진 않으신가요?

간편 본인인증을 하며 개인정보 유출에 대한 걱정 하신 적 있으신가요?

걱정마세요. 

블록체인과 AI를 활용한 본인인증 솔루션, `blockAi` 가 도와드릴게요!

## 🖥️ 기술스택
**Backend** `Java` `SpringBoot` `JPA`

**Frontend**  `Javascript` `Vue.js`

**Blockchain** `Solidity` `Remix`
